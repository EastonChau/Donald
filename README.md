## Trump Social Media vs. US Market Indices
Final Year Project (FYP)
This repository contains the code, data, and analysis for my final year project, which explores the statistical correlation between Donald Trump's social media activity and the performance of major US stock market indices.
## 📌 Project Overview
The goal of this research is to determine if sentiment, frequency, or specific keywords in social media posts (Truth Social, Twitter/X) act as leading indicators for market volatility or price movement in key US benchmarks.
## 📊 Key Indices Tracked

* S&P 500 (SPX)
* Dow Jones Industrial Average (DJI)
* NASDAQ Composite (IXIC)


## 🛠 Features

* Sentiment Analysis: Natural Language Processing (NLP) to quantify the tone of each post.
* Market Data Integration: Financial data fetched via APIs (e.g., Yahoo Finance, Bloomberg).
* Statistical Modeling: Correlation matrices, Granger causality tests, and time-series forecasting.
